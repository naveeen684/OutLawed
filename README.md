# website-react

To run the code:
  1)open a terminal
  1)git clone
  2)cd frontend\gui\
  3)npm install
  4)npm start
  5)open another terminal
  6)cd outlawed\backend\
  7)pip install requirements.txt
  8)cd src
  9)python manage.py runserver


Note: Both terminals needs to run parallely.
